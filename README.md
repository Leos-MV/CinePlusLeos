# CinePlusLeos
CinePlus es un prototipo funcional de un sistema de punto de venta de un cine desarrollado por en el lenguaje Python, con wxGlade para el desarrollo visual de ventanas y con VSCode para la edición de código.

El proyecto consta de cuatro clases, cada una con una ventana, la cuales son:
- Inicio de Sesión: Es la primera ventana y la que da el acceso a todo el programa.
- Taquilla: Es donde se selecciona la película, el horario y la sala, así como la cantidad y el tipo de boleto (para mayor o menor)
- Dulceria: Es donde se seleccionan los combos y los productos individuales
- Ticket: Se presenta la información de las ventanas anteriores y se procede con el cobro de la venta.

Para poder ejecutar el programa luego de haber descargado la carpeta con los archivos, primero se debe de contar con el lenguaje Python instalado, con un visualizador de código (para este ejemplo será VSCode) y la librería de Python wx descargada.
1. Abrir VSCode
2. Abrir el apartado "File"
3. Seleccionar "Open folder"
4. Buscar la carpeta con los archivos
5. Dar click en "Select folder"
6. Seleccionar el archivo "cineplus_v3.py"
7. Presionar el botón de "Play" para ejecutar el archivo
